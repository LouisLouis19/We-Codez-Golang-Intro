Overview

This section focuses on using `for` loops to iterate over sequences in Python.

For Loops

Basic structure:

for item in sequence:
    # Code to execute for each item

Example:

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

Using `range()` in For Loops

Example with `range()`:

for i in range(1, 6):
    print(i)

Break and Continue Statements

Break Statement

for i in range(1, 10):
    if i == 5:
        break
    print(i)

Continue Statement

for i in range(1, 10):
    if i % 2 == 0:
        continue
    print(i)

Practice

Try out these exercises to reinforce your understanding:

1. Print Numbers from 1 to 10

Write a program that prints numbers from 1 to 10 using a `for` loop.

Solution 
        for i in range(1, 11):
            print(i)

2. Calculate the Sum of Numbers

Write a program that calculates the sum of all numbers from 1 to 100 using a `for` loop.

Solution 
        total_sum = 0
        for i in range(1, 101):
            total_sum += i
        print("Total sum:", total_sum)

3. Print Each Character in a String

Write a program that prints each character of a string entered by the user on a new line.

Solution 
        text = input("Enter a string: ")
        for char in text:
            print(char)

4. Count Down from 10 to 1

Write a program that counts down from 10 to 1 using a `for` loop.

Solution 
        for i in range(10, 0, -1):
            print(i)

5. List of Squares

Write a program that prints the square of each number from 1 to 5.

Solution 
        for i in range(1, 6):
            print(i ** 2)

6. Print Multiples of 3

Write a program that prints the multiples of 3 between 1 and 30.

Solution 
        for i in range(3, 31, 3):
            print(i)

7. Print a List of Names

Write a program that prints each name from a list of names.

Solution 
        names = ["Alice", "Bob", "Charlie", "David"]
        for name in names:
            print(name)
