Overview

This section focuses on using for loops in Go to iterate over sequences. A for loop allows you to execute a block of code repeatedly for each item in a collection. This is particularly useful when you want to perform the same action on multiple items without needing to manually write out each operation.

For Loops

Basic structure:

for i := 0; i < len(sequence); i++ {
    // Code to execute for each item
}

In this structure:

  • i: A variable that acts as the loop counter, which takes values starting from 0, incrementing until it reaches the length of the sequence.
  • sequence: The collection of items you want to iterate over, such as an array, slice, or string.

Example:

fruits := []string{"apple", "banana", "cherry"}

for _, fruit := range fruits {
    fmt.Println(fruit)
}

In this example, the for loop iterates through the elements of the fruits slice. During each iteration, the variable fruit takes the value of the current item, and fmt.Println(fruit) outputs that value to the console. The output will be: 

apple
banana
cherry

More Examples:

1. Iterating Over a String:

word := "hello"
for _, letter := range word {
    fmt.Println(string(letter))
}

This will output each letter in the word "hello":

h
e
l
l
o

2. Summing Numbers in a Slice:

numbers := []int{1, 2, 3, 4, 5}
total := 0

for _, number := range numbers {
    total += number
}

fmt.Println("Total:", total)

In this example, the for loop iterates over each number in the numbers slice, adding each number to the total variable. The final output will be: 

Total: 15

Conclusion

For loops are a powerful tool in Go for iterating over sequences. They simplify repetitive tasks and enhance code readability. Understanding how to use for loops effectively will enable you to handle collections of data with ease.

Using range in For Loops

The range keyword in Go provides an easy way to iterate over the elements of a slice or array. It returns both the index and the value, which can be useful in some situations.

Example with range:

for i, num := range numbers {
    fmt.Println(i, num)
}

This will print each index and value of the numbers slice: 

0 1
1 2
2 3
3 4
4 5

More Examples:

1. Using range to Calculate a Sum:

total_sum := 0
for _, i := range range(1, 11) {
    total_sum += i
}

fmt.Println("Total Sum:", total_sum)

Conclusion

The range keyword is a very versatile tool for looping through collections, allowing you to handle data structures more easily.

Practice

Try out these exercises to reinforce your understanding:

1. Print Numbers from 1 to 10

Write a program that prints numbers from 1 to 10 using a `for` loop.

Solution 
for i := 1; i <= 10; i++ {
    fmt.Println(i)
}

2. Calculate the Sum of Numbers

Write a program that calculates the sum of all numbers from 1 to 100 using a `for` loop.

Solution 
total_sum := 0
for i := 1; i <= 100; i++ {
    total_sum += i
}
fmt.Println("Total sum:", total_sum)

3. Print Each Character in a String

Write a program that prints each character of a string (text)

text := "programming"
Solution 
text := "programming"
for _, char := range text {
    fmt.Println(string(char))
}

4. Count Down from 10 to 1

Write a program that counts down from 10 to 1 using a `for` loop.

Solution 
for i := 10; i >= 1; i-- {
    fmt.Println(i)
}

5. List of Squares

Write a program that prints the square of each number from 1 to 5.

Solution 
for i := 1; i <= 5; i++ {
    fmt.Println(i * i)
}

6. Print Multiples of 3

Write a program that prints the multiples of 3 between 1 and 30.

Solution 
for i := 3; i <= 30; i += 3 {
    fmt.Println(i)
}
Alternative Solution 
for i := 1; i <= 30; i++ {
    if i % 3 == 0 {
        fmt.Println(i)
    }
}

7. Print a List of Names

Write a program that prints each name from a list of names.

Solution 
names := []string{"Alice", "Bob", "Charlie"}
for _, name := range names {
    fmt.Println(name)
}
