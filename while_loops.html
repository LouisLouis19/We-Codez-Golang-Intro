While Loops

A while loop in Go repeatedly executes a block of code as long as a specified condition remains true. Unlike for loops, which iterate over a predefined sequence of elements, while loops are particularly useful when the number of iterations is not predetermined or when you want to keep looping until a specific condition is met.

The basic structure of a while loop in Go is as follows: 

for condition {
    // Code to execute while the condition is true
}

In a while loop:

  • Condition: The loop continues as long as the condition remains true. If the condition is false when the loop starts, the code inside the loop will not execute at all.
  • Loop Body: Contains the code that runs on each iteration as long as the condition is true.
  • Update Condition: It's important to ensure the condition eventually becomes false to prevent an infinite loop. This is usually done by updating variables within the loop.

Let’s look at a basic example in Go:

package main
import "fmt"

func main() {
    counter := 0

    for counter < 5 {
        fmt.Println("Counter is at:", counter)
        counter++
    }
}

In this example, the loop starts with counter set to 0. With each iteration, it prints the current value of counter and then increments it by 1. The loop stops once counter reaches 5, as the condition counter < 5 becomes false.

Be Careful with Infinite Loops

While loops are powerful tools that allow you to repeat a block of code as long as a specified condition remains true. However, if that condition never becomes false, the loop will continue indefinitely. This is known as an "infinite loop," which can cause your program to freeze, crash, or become unresponsive.

What Causes Infinite Loops?

Infinite loops typically happen when the loop condition is always true and never changes within the loop. This can result from a logic error or forgetting to update a variable that impacts the loop condition.

Example of an infinite loop in Go:

package main
import "fmt"

func main() {
    for {
        fmt.Println("This loop will run forever!")
    }
}

In this example, the condition for {} is constant and never becomes false. As a result, the loop will keep printing indefinitely. Such loops can exhaust your system’s resources, making your program unresponsive.

Practice Problems

1. Sum of Numbers

Write a program that calculates the sum of all numbers from 1 to 100. Solve this using both a for loop and a while loop (in Go).

Solution (Using for Loop) 
package main
import "fmt"

func main() {
    total_sum := 0
    for i := 1; i <= 100; i++ {
        total_sum += i
    }
    fmt.Println("Sum of numbers from 1 to 100 (for loop):", total_sum)
}
Solution (Using while Loop) 
package main
import "fmt"

func main() {
    total_sum := 0
    number := 1

    for number <= 100 {
        total_sum += number
        number++
    }
    fmt.Println("Sum of numbers from 1 to 100 (while loop):", total_sum)
}

2. Count Down from 10

Write a program that counts down from 10 to 1 using a while loop. The program should print each number in the countdown, followed by "Blast off!" when it reaches 0.

Solution 
package main
import "fmt"

func main() {
    count := 10

    for count > 0 {
        fmt.Println(count)
        count--
    }

    fmt.Println("Blast off!")
}

3. While Loop Factorial Calculation

Write a `while` loop that calculates the factorial of a given number. 

number := 6
Solution 
package main
import "fmt"

func main() {
    number := 6
    factorial := 1

    for number > 0 {
        factorial *= number
        number--
    }
    fmt.Println("Factorial:", factorial)
}

4. FizzBuzz

Write a program that prints the numbers from 1 to 50. For multiples of 3, print "Fizz" instead of the number, and for multiples of 5, print "Buzz". For numbers which are multiples of both 3 and 5, print "FizzBuzz".

Solution 
package main
import "fmt"

func main() {
    for i := 1; i <= 50; i++ {
        if i % 3 == 0 && i % 5 == 0 {
            fmt.Println("FizzBuzz")
        } else if i % 3 == 0 {
            fmt.Println("Fizz")
        } else if i % 5 == 0 {
            fmt.Println("Buzz")
        } else {
            fmt.Println(i)
        }
    }
}

5. Count the Number of Digits in a Number

Write a program that uses a while loop to count how many digits are in a given number (e.g., 12345 has 5 digits).

Solution 
package main
import "fmt"

func main() {
    number := 12345
    count := 0

    for number > 0 {
        count++
        number /= 10
    }

    fmt.Println("Number of digits:", count)
}
