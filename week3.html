Overview

This week, we will delve into `while` loops and offer extensive practice problems to reinforce your understanding of loops and conditional statements.

While Loops

A `while` loop in Python continues to execute a block of code as long as a specified condition remains true. Unlike `for` loops, `while` loops are ideal for scenarios where the number of iterations isn't predetermined.

Basic structure:

while condition:
    # Code to execute while the condition is True

Example:

counter = 0

while counter < 5:
    print("Counter is at:", counter)
    counter += 1

In this example, the loop continues to print the value of `counter` until it reaches 5, at which point the condition `counter < 5` becomes false, and the loop stops.

Be Careful with Infinite Loops

While loops are powerful tools that allow you to repeat a block of code as long as a condition is true. However, if that condition never becomes false, the loop will continue indefinitely. This is known as an "infinite loop." Infinite loops can cause your program to freeze, crash, or become unresponsive.

Example of an infinite loop:

while True:
print("This loop will run forever!")

In this case, because the condition True never changes to False, the loop runs forever. If you're not careful, infinite loops like this can lead to memory exhaustion or make your program hang.

How to Avoid Infinite Loops

To prevent infinite loops, you need to ensure that the loop's condition will eventually become false. This can be achieved in a few ways:

  • Modify a variable that affects the loop's condition inside the loop.
  • Use logical conditions that will eventually stop being true.
  • Use a break statement to forcibly exit the loop when a certain condition is met.

Example of a correct loop with a variable that affects the loop's condition: 

counter = 0
while counter < 5:
    print("Counter is", counter)
    counter += 1  # The condition will eventually become False

The break Statement

The break statement allows you to exit a loop prematurely, even if the loop’s condition is still true. This is particularly useful for situations where you are waiting for a specific condition to be met, or to prevent an infinite loop from continuing under certain circumstances.

Example of using the break statement to prevent an infinite loop: 

while True:
user_input = input("Type 'exit' to stop the loop: ")
if user_input == 'exit':
    print("Exiting the loop.")
    break  # Exit the loop if the user types 'exit'
else:
    print("The loop continues...")

In this case, the loop will only continue as long as the user doesn’t type "exit". When the user types "exit", the break statement stops the loop, even though the condition True would normally keep it going forever.

Tips for Preventing Infinite Loops

Here are a few best practices to avoid infinite loops in your programs:

  • Make sure the loop's condition is properly defined and will eventually be met.
  • Regularly check the loop's condition to ensure it can be changed by something inside the loop.
  • If you’re not sure about a loop’s behavior, use print statements to debug and see the loop’s progress.
  • Consider using the break statement if you need to exit the loop under specific conditions.

Remember, careful planning of your loop's logic will help you avoid infinite loops and keep your programs running smoothly.

Practice Problems

1. Sum of Numbers

Write a `for` loop that calculates the sum of all numbers from 1 to 100.

Solution 
total_sum = 0

for number in range(1, 101):
    total_sum += number

print("Sum of numbers from 1 to 100:", total_sum)

2. Guessing Game

Write a program that asks the user to guess a number between 1 and 10. The program should continue to ask until the user guesses the correct number (e.g., 7).

Solution 
correct_number = 7
guess = None

while guess != correct_number:
    guess = int(input("Guess a number between 1 and 10: "))
    if guess == correct_number:
        print("Congratulations! You guessed it right.")
    else:
        print("Try again!")

3. While Loop Factorial Calculation

Write a `while` loop that calculates the factorial of a given number.

Solution 
n = int(input("Enter a number: "))
factorial = 1

while n > 0:
    factorial *= n
    n -= 1

print("Factorial:", factorial)

4. FizzBuzz

Write a program that prints the numbers from 1 to 50. For multiples of 3, print "Fizz" instead of the number, and for multiples of 5, print "Buzz". For numbers which are multiples of both 3 and 5, print "FizzBuzz".

Solution 
for i in range(1, 51):
    if i % 3 == 0 and i % 5 == 0:
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)

5. Reverse a String

Write a `while` loop that reverses a given string.

Solution 
string = input("Enter a string: ")
reversed_string = ""
index = len(string) - 1

while index >= 0:
    reversed_string += string[index]
    index -= 1

print("Reversed string:", reversed_string)
