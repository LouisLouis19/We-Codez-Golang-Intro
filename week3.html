Overview

This week, we will delve into `while` loops and offer extensive practice problems to reinforce your understanding of loops and conditional statements.

While Loops

A `while` loop in Python continues to execute a block of code as long as a specified condition remains true. Unlike `for` loops, `while` loops are ideal for scenarios where the number of iterations isn't predetermined.

Basic structure:

while condition:
    # Code to execute while the condition is True

Example:

counter = 0

while counter < 5:
    print("Counter is at:", counter)
    counter += 1

In this example, the loop continues to print the value of `counter` until it reaches 5, at which point the condition `counter < 5` becomes false, and the loop stops.

Be Careful with Infinite Loops

While loops are powerful, but you must ensure that the loop's condition eventually becomes false. If it never does, the loop will continue indefinitely, causing an "infinite loop". This can crash your program or cause it to become unresponsive.

Example of an infinite loop:

while True:
    print("This loop will run forever!")

To prevent infinite loops, always ensure that the loop's condition will eventually become false, either by changing a variable within the loop or by using a break statement.

Practice Problems

1. For Loop: Sum of Numbers

Write a `for` loop that calculates the sum of all numbers from 1 to 100.

Solution 
total_sum = 0

for number in range(1, 101):
    total_sum += number

print("Sum of numbers from 1 to 100:", total_sum)

2. While Loop: Guessing Game

Write a program that asks the user to guess a number between 1 and 10. The program should continue to ask until the user guesses the correct number (e.g., 7).

Solution 
correct_number = 7
guess = None

while guess != correct_number:
    guess = int(input("Guess a number between 1 and 10: "))
    if guess == correct_number:
        print("Congratulations! You guessed it right.")
    else:
        print("Try again!")

3. If/Else: Even or Odd

Write a program that checks whether a given number is even or odd using an `if/else` statement.

Solution 
number = int(input("Enter a number: "))

if number % 2 == 0:
    print("The number is even.")
else:
    print("The number is odd.")

4. While Loop: Factorial Calculation

Write a `while` loop that calculates the factorial of a given number.

Solution 
n = int(input("Enter a number: "))
factorial = 1

while n > 0:
    factorial *= n
    n -= 1

print("Factorial:", factorial)

5. For Loop with If/Else: FizzBuzz

Write a program that prints the numbers from 1 to 50. For multiples of 3, print "Fizz" instead of the number, and for multiples of 5, print "Buzz". For numbers which are multiples of both 3 and 5, print "FizzBuzz".

Solution 
for i in range(1, 51):
    if i % 3 == 0 and i % 5 == 0:
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)

6. While Loop: Reverse a String

Write a `while` loop that reverses a given string.

Solution 
string = input("Enter a string: ")
reversed_string = ""
index = len(string) - 1

while index >= 0:
    reversed_string += string[index]
    index -= 1

print("Reversed string:", reversed_string)

7. If/Else: Age Classification

Write a program that categorizes a person's age into "Child," "Teenager," "Adult," or "Senior" based on user input.

Solution 
age = int(input("Enter your age: "))

if age < 13:
    print("You are a Child.")
elif 13 <= age < 20:
    print("You are a Teenager.")
elif 20 <= age < 65:
    print("You are an Adult.")
else:
    print("You are a Senior.")

8. For Loop: Sum of Even Numbers

Write a `for` loop that calculates the sum of all even numbers from 1 to 50.

Solution 
total_sum = 0

for number in range(1, 51):
    if number % 2 == 0:
        total_sum += number

print("Sum of even numbers from 1 to 50:", total_sum)

9. While Loop: Countdown Timer

Write a `while` loop that counts down from a given number to zero and prints "Liftoff!" when finished.

Solution 
countdown = int(input("Enter the countdown start number: "))

while countdown >= 0:
    print(countdown)
    countdown -= 1

print("Liftoff!")

10. For Loop: Multiplication Table

Write a `for` loop that prints the multiplication table of a given number (from 1 to 10).

Solution 
n = int(input("Enter a number: "))

for i in range(1, 11):
    print(n, "x", i, "=", n * i)
