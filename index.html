Overview

This week, we will focus on the basics of programming, the foundational rules of Go, and core concepts like variables, arithmetic operations, and logical operators.

Programming Basics

Programming involves writing instructions that a computer can execute. These instructions are written in a programming language, such as Go.

Core concepts include:

  • Syntax: The set of rules that define the structure of a programming language.
  • Variables: Containers for storing data values.
  • Data Types: The classification of data items, such as integers, floats, strings, and booleans.

Go Basics & Rules

Go is known for its simplicity, speed, and powerful concurrency features. Here are some key rules to follow when writing Go code:

  • Go code is case-sensitive.
  • Use indentation (not braces) to define code blocks (e.g., for loops or conditionals).
  • Comments in Go start with `//` for single-line comments and `/* */` for multi-line comments.
  • Go files typically have a `.go` extension.

Example of a simple Go code:

// This is a comment
package main
import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

Variables

Variables are like containers for storing data values. In Go, you can declare variables using the `var` keyword or short declaration `:=`.

Example of variable declaration and usage:

package main
import "fmt"

func main() {
    var myName = "Louis"
    fmt.Println("I am " + myName)
}

Here, `myAge` is a variable that stores the number `22`. The `fmt.Println` function is used to display the value of `myAge` along with some text.

Go Data Types

Understanding data types is crucial for writing efficient code. Here are the primary data types in Go:

  • Integer: Whole numbers, e.g., `42`
  • Float: Decimal numbers, e.g., `3.14`
  • String: Text data, e.g., `"Hello, World!"`
  • Boolean: True or False values, e.g., `true`
  • Array: Fixed-size collection of items, e.g., `[1, 2, 3]`
  • Map: Key-value pairs, e.g., `{"name": "Alice", "age": 25}`

Examples:

// Integer
age := 22

// Float
height := 5.9

// String
greeting := "Hello, World!"

// Boolean
isStudent := true

// Array
fruits := [3]string{"apple", "banana", "cherry"}

// Map
person := map[string]interface{}{"name": "Alice", "age": 25}

Arithmetic Operators

Arithmetic operations allow you to perform calculations using numbers. Go supports basic arithmetic operators:

  • Addition: `+`
  • Subtraction: `-`
  • Multiplication: `*`
  • Division: `/`
  • Modulus: `%`

Examples:

// Addition
sum := 5 + 3
fmt.Println(sum)
// Subtraction
difference := 10 - 4
fmt.Println(difference)
// Multiplication
product := 7 * 6
fmt.Println(product)
// Division
quotient := 20 / 4
fmt.Println(quotient)
// Modulus: Remainder after Division
remainder := 10 % 3
fmt.Println(remainder)

Logical (Comparison) Operators

Comparison operators allow you to compare two values and return a Boolean result (`true` or `false`). They are essential in decision-making within your code.

  • Equal To: `==`
  • Not Equal To: `!=`
  • Greater Than: `>`
  • Less Than: `<`
  • Greater Than or Equal To: `>=`
  • Less Than or Equal To: `<=`

Examples:

// Equal To
result := 5 == 5 // true

// Not Equal To
result := 5 != 5 // false

// Greater Than
result := 7 > 5 // true

// Less Than
result := 5 < 6 // true

// Greater Than or Equal To
result := 5 >= 5 // true

// Less Than or Equal To
result := 7 <= 5 // false

fmt.Println(result)

Reading Input from Users

In Go, you can use the `fmt.Scan` function to read input from users.

Basic Usage

To read input from the user, use the `fmt.Scan` function. Here is a basic example: 

package main
import "fmt"

func main() {
    var userName string
    fmt.Print("Enter your name: ")
    fmt.Scan(&userName)
    fmt.Println("Hello, " + userName + "!")
}

Reading Numeric Input

If you need to read numeric input, you'll need to use the `fmt.Scan` function with the appropriate data type. Here is an example: 

package main
import "fmt"

func main() {
    var userAge int
    fmt.Print("Enter your age: ")
    fmt.Scan(&userAge)
    fmt.Println("You are", userAge, "years old.")
}
