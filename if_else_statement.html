Overview

This section introduces conditional statements in Go, with a focus on the if/else structure. Conditional statements are essential for decision-making in programming, allowing your code to perform different actions based on specific conditions. By understanding and using if and else statements effectively, you can make your programs more dynamic and responsive.

If/Else Statements

Conditional statements enable your program to make decisions and execute certain blocks of code depending on whether specific conditions are met. The if/else structure is one of the most common ways to create conditional logic, making it a fundamental tool for controlling program flow.

Basic Structure

The syntax for an if/else statement in Go is simple: 

if condition {
    // Code to execute if the condition is true
} else {
    // Code to execute if the condition is false
}
  • If: The if keyword introduces the condition. If the condition evaluates to true, the code inside the if block executes.
  • Else: The else block is optional and only runs if the if condition is false. If there’s no else block, the program skips it and moves on.
  • Condition: The condition is typically an expression that evaluates to a boolean value (true or false). Go will evaluate the condition, and only one of the blocks will execute based on whether the condition is met.

Example

Let’s look at a simple example to demonstrate the if/else structure: 

package main
import "fmt"

func main() {
    age := 20

    if age >= 18 {
        fmt.Println("You are an adult.")
    } else {
        fmt.Println("You are a minor.")
    }
}

In this example, the program checks if the variable age is 18 or older. If the condition is true, it prints "You are an adult." If it’s false, it prints "You are a minor."

Nested If/Else Statements

You can nest if/else statements within each other to handle more complex conditions. This structure is especially helpful when decisions depend on multiple related factors. 

package main
import "fmt"

func main() {
    age := 17
    hasPermission := true

    if age >= 18 {
        fmt.Println("You are an adult.")
    } else {
        if hasPermission {
            fmt.Println("You are a minor, but you have permission.")
        } else {
            fmt.Println("You are a minor and do not have permission.")
        }
    }
}

In this example, if age is under 18, a second if statement checks whether hasPermission is true. This nested approach allows for more detailed decision-making.

Common Pitfalls

  • Indentation Errors: Each block of code inside if, else statements must be consistently indented. Go uses indentation to separate code blocks.
  • Order of Conditions: In a series of if and else if statements, Go stops evaluating once it finds a true condition. Make sure conditions are ordered logically to avoid unintended results.

Else If Statements

The if statement allows us to execute code based on a condition. However, when you need to check multiple conditions, using just an if statement can be inefficient. This is where else if and else come into play.

Else if statements help handle multiple conditions. Once a condition in an if or else if block evaluates as true, the corresponding block of code is executed, and the remaining conditions are ignored.

The else block serves as a "catch-all" for situations where none of the if or else if conditions are met. It ensures that your code still responds to inputs even when they don't match any specific condition.

Example:

package main
import "fmt"

func main() {
    score := 85
    
    if score >= 90 {
        fmt.Println("Grade: A")
    } else if score >= 80 {
        fmt.Println("Grade: B")
    } else if score >= 70 {
        fmt.Println("Grade: C")
    } else {
        fmt.Println("Grade: F")
    }
}

Explanation: In this example, the program checks a score to determine the appropriate grade. The first condition checks if the score is greater than or equal to 90. If true, it prints "Grade: A". If false, it moves to the next else if statement, checking if the score is greater than or equal to 80, and so on. Finally, if none of the conditions are true, the else block runs, printing "Grade: F". Note that once a condition is true, the remaining conditions are skipped.

This conditional structure ensures flexibility in decision-making by providing multiple options for varying conditions. It also keeps your code clean and easy to follow, avoiding complex and nested if statements.

Logical Operators in Go

Logical operators allow you to combine multiple conditions in your code, enabling you to perform complex checks based on several criteria. Go has three primary logical operators:

  • AND
  • OR
  • NOT

AND Operator (&&)

The && operator checks if both conditions are true. If either condition is false, the entire expression will evaluate to false. 

package main
import "fmt"

func main() {
    age := 25
    if age >= 18 && age <= 35 {
        fmt.Println("You are a young adult.")
    }
}

Explanation: The condition age >= 18 && age <= 35 will only be true if age is both 18 or older and 35 or younger. If age were 40, this statement would not print anything because age <= 35 would be false.

More Examples with AND

package main
import "fmt"

func main() {
    salary := 5000
    yearsExperience := 3
    if salary > 3000 && yearsExperience > 2 {
        fmt.Println("Eligible for senior position.")
    }
}

This example checks that the person has a salary greater than 3000 and more than 2 years of experience.

OR Operator (||)

The || operator checks if at least one condition is true. If any one of the conditions is true, the entire expression will evaluate to true. 

package main
import "fmt"

func main() {
    temperature := 30
    if temperature < 15 || temperature > 25 {
        fmt.Println("The weather is outside the comfortable range.")
    }
}

Explanation: This code checks if the temperature is below 15 or above 25. If either condition is true, the message will print.

More Examples with OR

package main
import "fmt"

func main() {
    day := "Saturday"
    if day == "Saturday" || day == "Sunday" {
        fmt.Println("It's the weekend!")
    }
}

This example checks if the day is either Saturday or Sunday to determine if it's the weekend.

NOT Operator (!)

The ! operator reverses the Boolean value of a condition. If the condition is true, ! makes it false, and if it’s false, ! makes it true. 

package main
import "fmt"

func main() {
    isRaining := false
    if !isRaining {
        fmt.Println("You can leave the umbrella at home.")
    }
}

Explanation: !isRaining inverts the value of isRaining. So if isRaining is false, !isRaining is true, and the message will print.

More Examples with NOT

package main
import "fmt"

func main() {
    isSunny := true
    if !isSunny {
        fmt.Println("It might be cloudy or rainy today.")
    }
}

This example checks if isSunny is false (i.e., it’s not sunny) and prints a message accordingly.

Summary Table

Operator Description Example Result
&& True if both conditions are true True && False False
|| True if at least one condition is true True || False True
! Reverses the Boolean value of a single condition !True False

        fmt.Println("Negative")
    }
}
