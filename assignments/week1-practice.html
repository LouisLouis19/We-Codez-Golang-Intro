3. Correct the Arithmetic Operation

Given the following code, identify and fix any errors related to arithmetic operations:

number = 10
result = number / 0
print(result)
Solution 
# Corrected code
number = 10
result = number / 2  # Changed from 0 to 2 to avoid division by zero
print(result)

4. User Input - Favorite Number

Write a program that asks the user for their favorite number, doubles it, and prints the result.

Solution 
favorite_number = int(input("Enter your favorite number: "))
doubled_number = favorite_number * 2
print("Double your favorite number is " + str(doubled_number))

5. User Input - Greeting

Create a program that asks the user for their name and then prints a greeting message with their name.

Solution 
user_name = input("Enter your name: ")
print("Hello, " + user_name + "!")

6. Calculate the Area of a Circle

Write a program that takes a radius (number) from the user, calculate the area of a circle with the given radius, and print the area.

Solution 
radius = float(input("Enter a radius: "))
area = 3.14 * radius * radius
print("The area of circle is " + str(area))

7. Convert Fahrenheit to Celsius

Write a program that converts a temperature from Fahrenheit to Celsius. Use the formula: Celsius = (Fahrenheit - 32) * 5/9.

Solution 
fahrenheit = float(input("Enter temperature in Fahrenheit: "))
celsius = (fahrenheit - 32) * 5/9
print("Temperature in Celsius is " + str(celsius))
