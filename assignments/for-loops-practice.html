Python Practice Exercises - For Loops

Complete the following exercises to strengthen your understanding of For Loops Concepts.

1. Generate Multiples of a Fixed Number

Write a program that prints the first 10 multiples of the number 5 using a `for` loop. 

number = 5
Solution 
number = 5
for i in range(1, 11):
    print(number * i)

2. Sum of Even Numbers Up to 20

Write a program that calculates the sum of all even numbers between 1 and 20 using a `for` loop.

Solution 
limit = 20
sum_even = 0
for i in range(2, limit + 1, 2):
    sum_even += i
print("Sum of even numbers:", sum_even)

3. Fibonacci Sequence

Write a program that prints the first 10 numbers of the Fibonacci sequence using a `for` loop.

Solution 
a, b = 0, 1
print(a)
for _ in range(9):
    a, b = b, a + b
    print(a)

4. Count Numbers Greater Than 10

Write a program that counts and prints the number of elements in a list that are greater than 10. 

numbers = [4, 12, 7, 15, 9, 20, 3]
Solution 
numbers = [4, 12, 7, 15, 9, 20, 3]
count = 0

for num in numbers:
    if num > 10:
        count += 1

print("Count of numbers greater than 10:", count)

5. Find the Maximum Number in a List

Write a program that finds and prints the maximum number from the list [10, 25, 15, 30, 5].

Solution 
  numbers = [10, 25, 15, 30, 5]
  max_num = numbers[0]
  for num in numbers:
      if num > max_num:
          max_num = num
  print("The maximum number is:", max_num)

6. Calculate the Sum of a List of Numbers

Write a program that calculates and prints the sum of the numbers in a list. 

numbers = [1, 2, 3, 4, 5]
Solution 
numbers = [1, 2, 3, 4, 5]
total_sum = 0
for num in numbers:
    total_sum += num
print("The sum of the numbers is:", total_sum)

7. Factorial of a Number

Write a program that calculates the factorial of a number using a `for` loop. 

number = 5
Solution 
number = 5
factorial = 1
for i in range(1, number + 1):
    factorial *= i
print("Factorial:", factorial)

8. Count Odd and Even Numbers

Write a program that counts the number of odd and even numbers in a list using `for` loop. 

numbers = [3, 8, 12, 15, 7, 10, 21, 14]
Solution 
numbers = [3, 8, 12, 15, 7, 10, 21, 14]
count_odd = 0
count_even = 0

for num in numbers:
    if num % 2 == 0:
        count_even += 1
    else:
        count_odd += 1

print("Count of even numbers:", count_even)
print("Count of odd numbers:", count_odd)

9. Sum of Odd Numbers

Write a program that calculates the sum of all odd numbers between 1 and a number using a `for` loop. 

number = 10
Solution 
number = 10
sum_odd = 0
for i in range(1, number + 1, 2):
    sum_odd += i
print("Sum of odd numbers:", sum_odd)

9. Count Numbers Greater Than Their Previous

Write a program that counts how many times a number in a list that is greater than the number that came before it. 

numbers = [5, 3, 8, 6, 10, 15, 7, 18]]
Solution 
numbers = [5, 3, 8, 6, 10, 15, 7, 18]
count = 0

for i in range(1, len(numbers)):
    if numbers[i] > numbers[i - 1]:
        count += 1

print("Count of numbers greater than their previous:", count)
