Python Practice Exercises - For Loops

Complete the following exercises to strengthen your understanding of for loops.

1. Generate Multiples of a Number

Write a program that takes a number from the user and prints its first 10 multiples using a `for` loop.

Solution 
number = int(input("Enter a number: "))
for i in range(1, 11):
    print(number * i)

2. Sum of Even Numbers

Write a program that calculates the sum of all even numbers between 1 and a user-defined number using a `for` loop.

Solution 
limit = int(input("Enter a limit: "))
sum_even = 0
for i in range(2, limit + 1, 2):
    sum_even += i
print("Sum of even numbers:", sum_even)

3. Fibonacci Sequence

Write a program that prints the first 10 numbers of the Fibonacci sequence using a `for` loop.

Solution 
a, b = 0, 1
print(a, end=' ')
for _ in range(9):
    a, b = b, a + b
    print(a, end=' ')
print()

4. Print a Pyramid Pattern

Write a program that prints a pyramid pattern of stars. The number of rows should be taken from the user.

Solution 
rows = int(input("Enter number of rows: "))
for i in range(1, rows + 1):
    print(' ' * (rows - i) + '*' * (2 * i - 1))

5. Count Vowels in a String

Write a program that counts the number of vowels in a user-provided string using a `for` loop.

Solution 
text = input("Enter a string: ")
vowels = 'aeiouAEIOU'
count = 0
for char in text:
    if char in vowels:
        count += 1
print("Number of vowels:", count)

6. Reverse a String

Write a program that takes a string input from the user and prints the string in reverse order using a `for` loop.

Solution 
text = input("Enter a string: ")
reversed_text = ""
for char in text:
    reversed_text = char + reversed_text
print("Reversed string:", reversed_text)

7. Factorial of a Number

Write a program that takes a number from the user and calculates its factorial using a `for` loop.

Solution 
number = int(input("Enter a number: "))
factorial = 1
for i in range(1, number + 1):
    factorial *= i
print("Factorial:", factorial)

8. Sum of Digits

Write a program that takes an integer input from the user and calculates the sum of its digits using a `for` loop.

Solution 
number = input("Enter a number: ")
sum_digits = 0
for digit in number:
    sum_digits += int(digit)
print("Sum of digits:", sum_digits)

9. Create a Multiplication Table

Write a program that takes a number from the user and prints a multiplication table up to 10 using a `for` loop.

Solution 
number = int(input("Enter a number: "))
for i in range(1, 11):
    print(f"{number} x {i} = {number * i}")

10. Sum of Odd Numbers

Write a program that calculates the sum of all odd numbers between 1 and a user-defined number using a `for` loop.

Solution 
limit = int(input("Enter a limit: "))
sum_odd = 0
for i in range(1, limit + 1, 2):
    sum_odd += i
print("Sum of odd numbers:", sum_odd)
