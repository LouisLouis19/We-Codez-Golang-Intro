Complete the following exercises to strengthen your understanding of Python Function Concepts.

1. Sum of a List

Create a function called `sum_of_list` that takes a list of numbers as a parameter and returns the sum of all numbers in the list using a for loop.

Solution 
def sum_of_list(numbers):
    total = 0
    for num in numbers:
        total += num
    return total

print(sum_of_list([1, 2, 3, 4, 5]))  # Output: 15

2. Factorial of a Number

Create a function called `factorial` that takes an integer as a parameter and returns the factorial of that number using a while loop.

Solution 
def factorial(n):
    result = 1
    while n > 1:
        result *= n
        n -= 1
    return result

print(factorial(5))  # Output: 120

3. Count Vowels in a String

Create a function called `count_vowels` that takes a string as a parameter and returns the number of vowels (a, e, i, o, u) in the string.

Solution 
def count_vowels(s):
    vowels = "aeiouAEIOU"
    count = 0
    for char in s:
        if char in vowels:
            count += 1
    return count

print(count_vowels("Hello World"))  # Output: 3

4. Reverse a String

Create a function called `reverse_string` that takes a string as a parameter and returns the string reversed.

Solution 
def reverse_string(s):
    reversed_str = ""
    for char in s:
        reversed_str = char + reversed_str
    return reversed_str

print(reverse_string("Hello"))  # Output: "olleH"

5. Sum of Even Numbers in a Range

Create a function called `sum_of_evens` that takes two integers as parameters (start and end) and returns the sum of all even numbers in that range using a for loop.

Solution 
def sum_of_evens(start, end):
    total = 0
    for num in range(start, end + 1):
        if num % 2 == 0:
            total += num
    return total

print(sum_of_evens(1, 10))  # Output: 30

6. Find Common Elements in Two Lists

Create a function called `common_elements` that takes two lists as parameters and returns a list of common elements between the two lists.

Solution 
def common_elements(list1, list2):
    common = []
    for element in list1:
        if element in list2:
            common.append(element)
    return common

print(common_elements([1, 2, 3], [2, 3, 4]))  # Output: [2, 3]

7. Prime Number Checker

Create a function called `is_prime` that takes an integer as a parameter and returns `True` if the number is prime, and `False` otherwise.

Solution 
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True

print(is_prime(11))  # Output: True
print(is_prime(10))  # Output: False

8. Palindrome Checker

Create a function called `is_palindrome` that takes a string as a parameter and returns `True` if the string is a palindrome (reads the same forward and backward), and `False` otherwise.

Solution 
def is_palindrome(s):
    return s == s[::-1]

print(is_palindrome("racecar"))  # Output: True
print(is_palindrome("hello"))    # Output: False

9. Remove Duplicates from a List

Create a function called `remove_duplicates` that takes a list as a parameter and returns a new list with duplicates removed, preserving the original order.

Solution 
def remove_duplicates(lst):
    unique = []
    for item in lst:
        if item not in unique:
            unique.append(item)
    return unique

print(remove_duplicates([1, 2, 2, 3, 4, 4, 5]))  # Output: [1, 2, 3, 4, 5]

10. Count Occurrences of an Element in a List

Create a function called `count_occurrences` that takes a list and an element as parameters and returns the number of occurrences of that element in the list.

Solution 
def count_occurrences(lst, element):
    count = 0
    for item in lst:
        if item == element:
            count += 1
    return count

print(count_occurrences([1, 2, 2, 3, 2], 2))  # Output: 3

11. Find Maximum of a List

Create a function called `max_of_list` that takes a list of numbers as a parameter and returns the largest number in the list using a for loop.

Solution 
def max_of_list(numbers):
    max_num = numbers[0]
    for num in numbers:
        if num > max_num:
            max_num = num
    return max_num

print(max_of_list([3, 5, 7, 2, 8]))  # Output: 8

13. Cumulative Sum of a List

Create a function called `cumulative_sum` that takes a list of numbers as a parameter and returns a list where each element is the cumulative sum of the numbers up to that index.

Solution 
def cumulative_sum(numbers):
    cum_sum = []
    total = 0
    for num in numbers:
        total += num
        cum_sum.append(total)
    return cum_sum

print(cumulative_sum([1, 2, 3, 4]))  # Output: [1, 3, 6, 10]
