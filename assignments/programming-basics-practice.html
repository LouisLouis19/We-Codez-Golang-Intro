4. User Input - Favorite Number

Write a program that asks the user for their favorite number, doubles it, and prints the result.

Solution 
favorite_number = int(input("Enter your favorite number: "))
doubled_number = favorite_number * 2
print("Double your favorite number is " + str(doubled_number))

user_name = input("Enter your name: ")

6. Calculate the Area of a Circle

Write a program that takes a radius (number) from the user, calculate the area of a circle with the given radius, and print the area.

Solution 
radius = float(input("Enter a radius: "))
area = 3.14 * radius * radius
print("The area of circle is " + str(area))

7. Convert Fahrenheit to Celsius

Write a program that converts a temperature from Fahrenheit to Celsius. Use the formula: Celsius = (Fahrenheit - 32) * 5/9.

Solution 
fahrenheit = float(input("Enter temperature in Fahrenheit: "))
celsius = (fahrenheit - 32) * 5/9
print("Temperature in Celsius is " + str(celsius))

8. Calculate the Perimeter of a Rectangle

Write a program that takes the length and width of a rectangle from the user, calculates the perimeter, and prints it.

Solution 
length = int(input("Enter the length of the rectangle: "))
width = int(input("Enter the width of the rectangle: "))
perimeter = 2 * (length + width)
print("The perimeter of the rectangle is " + str(perimeter))

9. Calculate the Volume of a Cube

Write a program that takes the side length of a cube from the user, calculates the volume, and prints it.

Solution 
side_length = int(input("Enter the side length of the cube: "))
volume = side_length ** 3
print("The volume of the cube is " + str(volume))

10. Convert Kilometers to Miles

Write a program that converts a distance from kilometers to miles. Use the formula: Miles = Kilometers * 0.621371.

Solution 
kilometers = float(input("Enter distance in kilometers: "))
miles = kilometers * 0.621371
print("Distance in miles is " + str(miles))

11. Calculate the Average of Three Numbers

Write a program that takes three numbers from the user, calculates their average, and prints the result.

Solution 
num1 = int(input("Enter the first number: "))
num2 = int(input("Enter the second number: "))
num3 = int(input("Enter the third number: "))
average = (num1 + num2 + num3) / 3
print("The average of the three numbers is " + str(average))
