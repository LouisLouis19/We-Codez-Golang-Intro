Go Practice Exercises

Complete the following exercises to strengthen your understanding of If-Else Concepts in Go

1. Simple If/Else Statement

Check if a number is positive or negative:

number := -5
Solution 
package main
import "fmt"
func main() {
    number := -5
    if number >= 0 {
        fmt.Println("The number is positive.")
    } else {
        fmt.Println("The number is negative.")
    }
}

2. Elif Statement

Determine the life stage of a person based on their age.

  • Child: age is less than 13
  • Teenager: age is between 13 and 19, inclusive
  • Adult: age is between 20 and 64, inclusive
  • Senior: age is 65 or older
age := 65
Solution 
package main
import "fmt"
func main() {
    age := 65
    if age < 13 {
        fmt.Println("Child")
    } else if age >= 13 && age < 20 {
        fmt.Println("Teenager")
    } else if age >= 20 && age < 65 {
        fmt.Println("Adult")
    } else {
        fmt.Println("Senior")
    }
}

3. Check Even or Odd

Write a program that checks if a number is even or odd:

number := 7
Solution 
package main
import "fmt"
func main() {
    number := 7
    if number%2 == 0 {
        fmt.Println("The number is even.")
    } else {
        fmt.Println("The number is odd.")
    }
}

4. Check Divisibility by 3, 5, or Both

Create a program that checks if a number is divisible by 3, 5, or both: 

number := 15
Solution 
package main
import "fmt"
func main() {
    number := 15
    if number%3 == 0 && number%5 == 0 {
        fmt.Println("Divisible by both 3 and 5.")
    } else if number%3 == 0 {
        fmt.Println("Divisible by 3.")
    } else if number%5 == 0 {
        fmt.Println("Divisible by 5.")
    } else {
        fmt.Println("Not divisible by 3 or 5.")
    }
}

5. Determine the Largest of Three Numbers

Write a program to find the largest of three numbers:

a, b, c := 5, 12, 7
Solution 
package main
import "fmt"
func main() {
    a, b, c := 5, 12, 7
    if a >= b && a >= c {
        fmt.Printf("The largest number is: %d\n", a)
    } else if b >= a && b >= c {
        fmt.Printf("The largest number is: %d\n", b)
    } else {
        fmt.Printf("The largest number is: %d\n", c)
    }
}

6. Check Leap Year

Write a program to check if a year is a leap year. A year is a leap year if:

  • It is divisible by 4, and not divisible by 100, or
  • It is divisible by 400.

Example:

year := 2020
Solution 
package main
import "fmt"
func main() {
    year := 2020
    if (year%4 == 0 && year%100 != 0) || (year%400 == 0) {
        fmt.Println("Leap year")
    } else {
        fmt.Println("Not a leap year")
    }
}

7. Check Temperature Range

Write a program that categorizes the temperature based on its value.

  • If the temperature is 30 degrees Celsius or above, print "It's hot."
  • If the temperature is between 20 (inclusive) and 30 (exclusive) degrees Celsius, print "It's warm."
  • If the temperature is between 10 (inclusive) and 20 (exclusive) degrees Celsius, print "It's cool."
  • If the temperature is below 10 degrees Celsius, print "It's cold." 
temperature := 30
Solution 
package main
import "fmt"
func main() {
    temperature := 30
    if temperature >= 30 {
        fmt.Println("It's hot.")
    } else if temperature >= 20 {
        fmt.Println("It's warm.")
    } else if temperature >= 10 {
        fmt.Println("It's cool.")
    } else {
        fmt.Println("It's cold.")
    }
}

8. Determine Zero, Odd or Even

Write a program that determines if a given number is zero, odd, or even. Print the result. 

number := 3
Solution 
package main
import "fmt"
func main() {
    number := 3
    if number == 0 {
        fmt.Println("The number is zero")
    } else if number%2 == 0 {
        fmt.Println("The number is even")
    } else {
        fmt.Println("The number is odd")
    }
}

9. Grade Classification

Write a program that reads a score and classifies it into grades: 'A' for scores 90 and above, 'B' for scores between 80 and 89, 'C' for scores between 70 and 79, and 'F' for scores below 70. 

score := 85
Solution 
package main
import "fmt"
func main() {
    score := 85
    if score >= 90 {
        fmt.Println("Grade: A")
    } else if score >= 80 {
        fmt.Println("Grade: B")
    } else if score >= 70 {
        fmt.Println("Grade: C")
    } else {
        fmt.Println("Grade: F")
    }
}
