1. Find Non-Adjacent Sum

Write a function non_adjacent_sum that takes a list of numbers and returns the sum of all non-adjacent numbers (numbers that do not directly follow each other in the list).

Solution 
def non_adjacent_sum(numbers):
    return sum(numbers[::2])

numbers = [1, 2, 3, 4, 5, 6]
print(non_adjacent_sum(numbers))  # Output: 9

2. Loop with Conditional Logic and Built-in Function

Create a function remove_duplicates that takes a list of numbers and removes any duplicate values while preserving the order of elements. Use a loop and a built-in function to handle this.

Solution 
def remove_duplicates(numbers):
    unique_numbers = []
    for num in numbers:
        if num not in unique_numbers:
            unique_numbers.append(num)
    return unique_numbers

numbers = [1, 2, 2, 3, 4, 4, 5]
print(remove_duplicates(numbers))  # Output: [1, 2, 3, 4, 5]

3. Find Maximum After Removing a Specific Value

Write a function max_after_removal that takes a list of numbers and a value to remove from the list. The function should return the maximum value in the list after the specified value has been removed.

Solution 
def max_after_removal(numbers, value_to_remove):
    numbers = [num for num in numbers if num != value_to_remove]
    return max(numbers)

numbers = [1, 3, 7, 8, 7, 5]
print(max_after_removal(numbers, 7))  # Output: 8

4. Loop with String Manipulation

Create a function replace_vowels that takes a string as a parameter and replaces all vowels with a specified character (default is '*'). Ensure the function works with both uppercase and lowercase letters.

Solution 
def replace_vowels(text, replacement='*'):
    vowels = "aeiouAEIOU"
    for vowel in vowels:
        text = text.replace(vowel, replacement)
    return text

print(replace_vowels("Hello World"))  # Output: H*ll* W*rld

5. Find the Difference Between Consecutive Elements

Write a function find_differences that takes a list of numbers and returns a list of the absolute differences between consecutive elements.

Solution 
def find_differences(numbers):
    differences = []
    for i in range(1, len(numbers)):
        differences.append(abs(numbers[i] - numbers[i - 1]))
    return differences

numbers = [10, 5, 12, 8, 15]
print(find_differences(numbers))  # Output: [5, 7, 4, 7]

6. Sum of List with Default Step Parameter

Create a function sum_list that takes a list of numbers and an optional step parameter (default value is 1). The function should return the sum of numbers in the list, stepping through the list according to the given step.

Solution 
def sum_list(numbers, step=1):
    return sum(numbers[::step])

numbers = [1, 2, 3, 4, 5, 6]
print(sum_list(numbers))      # Output: 21
print(sum_list(numbers, 2))   # Output: 9

7. Product of List with Default Exclusion

Write a function product_of_list that takes a list of numbers and an optional value to exclude from the product (default is None). The function should return the product of all numbers in the list that are not equal to the excluded value.

Solution 
def product_of_list(numbers, exclude=None):
    product = 1
    for num in numbers:
        if num != exclude:
            product *= num
    return product

numbers = [2, 3, 4, 5]
print(product_of_list(numbers))            # Output: 120
print(product_of_list(numbers, exclude=3)) # Output: 40

8. Reverse and Double List Elements

Write a function reverse_and_double that takes a list of numbers and returns a new list where the order of the numbers is reversed and each number is doubled.

Solution 
def reverse_and_double(numbers):
    result = []
    for num in numbers[::-1]:
        result.append(num * 2)
    return result

numbers = [1, 2, 3, 4]
print(reverse_and_double(numbers))  # Output: [8, 6, 4, 2]

9. Accumulate the Sum with Default Parameters

Write a function accumulate_sum that takes a list of numbers and an optional start index (default is 0) and returns the cumulative sum starting from the given index.

Solution 
def accumulate_sum(numbers, start=0):
    total = 0
    for num in numbers[start:]:
        total += num
    return total

numbers = [5, 10, 15, 20]
print(accumulate_sum(numbers))       # Output: 50
print(accumulate_sum(numbers, 2))    # Output: 35

10. Find Prime Numbers in a Range

Write a function find_primes_in_range that takes two integers, start and end, and returns a list of all prime numbers between these two values (inclusive). A prime number is a number greater than 1 that has no positive divisors other than 1 and itself.

Solution 
def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

def find_primes_in_range(start, end):
    primes = []
    for num in range(start, end + 1):
        if is_prime(num):
            primes.append(num)
    return primes

print(find_primes_in_range(10, 20))  # Output: [11, 13, 17, 19]
