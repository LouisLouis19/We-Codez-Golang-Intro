Complete the following exercises to strengthen your understanding of Python Function Concepts.

1. Sum of a List

Create a function called `sum_of_list` that takes a list of numbers as a parameter and returns the sum of all numbers in the list using a for loop.

Solution 
def sum_of_list(numbers):
    total = 0
    for num in numbers:
        total += num
    return total

print(sum_of_list([1, 2, 3, 4, 5]))  # Output: 15

2. Factorial of a Number

Create a function called `factorial` that takes an integer as a parameter and returns the factorial of that number using a while loop.

Solution 
def factorial(n):
    result = 1
    while n > 1:
        result *= n
        n -= 1
    return result

print(factorial(5))  # Output: 120

3. Check Even or Odd

Create a function called is_even that takes a number as a parameter and returns True if the number is even and False if it is odd. This task will reinforce your understanding of conditionals and function definitions.

Solution 
def is_even(num):

5. Sum of Even Numbers in a Range

Create a function called `sum_of_evens` that takes two integers as parameters (start and end) and returns the sum of all even numbers in that range using a for loop.

Solution 
def sum_of_evens(start, end):
    total = 0
    for num in range(start, end + 1):
        if num % 2 == 0:
            total += num
    return total

print(sum_of_evens(1, 10))  # Output: 30

7. Prime Number Checker

Create a function called `is_prime` that takes an integer as a parameter and returns `True` if the number is prime, and `False` otherwise.

Solution 
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True

print(is_prime(11))  # Output: True
print(is_prime(10))  # Output: False

8. Digit Sum Calculator

Create a function called digit_sum that takes a positive integer as a parameter and returns the sum of its digits. This exercise will help you practice using while loops to manipulate numbers.

Solution 
def digit_sum(n):
    total = 0
    while n > 0:
        total += n % 10  # Add the last digit to the total
        n //= 10         # Remove the last digit from n
    return total

# Example usage:
print(digit_sum(123))  # Output: 6 (1 + 2 + 3)
print(digit_sum(4567)) # Output: 22 (4 + 5 + 6 + 7)

9. Count Odd Digits in a Number

Create a function called count_odd_digits that takes a positive integer as a parameter and returns the count of odd digits in that number.

Solution 
def count_odd_digits(n):
    count = 0
    while n > 0:
        digit = n % 10  # Extract the last digit
        if digit % 2 != 0:  # Check if it's odd
            count += 1
        n //= 10  # Remove the last digit
    return count

# Example usage:
print(count_odd_digits(12345))  # Output: 3 (1, 3, 5 are odd)
print(count_odd_digits(2468))    # Output: 0 (no odd digits)

10. Count Occurrences of an Element in a List

Create a function called `count_occurrences` that takes a list and an element as parameters and returns the number of occurrences of that element in the list.

Solution 
def count_occurrences(lst, element):
    count = 0
    for item in lst:
        if item == element:
            count += 1
    return count

print(count_occurrences([1, 2, 2, 3, 2], 2))  # Output: 3

11. Find Maximum of a List

Create a function called `max_of_list` that takes a list of numbers as a parameter and returns the largest number in the list using a for loop.

Solution 
def max_of_list(numbers):
    max_num = numbers[0]
    for num in numbers:
        if num > max_num:
            max_num = num
    return max_num

print(max_of_list([3, 5, 7, 2, 8]))  # Output: 8

13. Find the Maximum Digit in a Number

Create a function called max_digit that takes a positive integer as a parameter and returns the maximum digit found in that number.

Solution 
def max_digit(n):
    max_digit = 0
    while n > 0:
        digit = n % 10  # Extract the last digit
        if digit > max_digit:  # Update max_digit if the current digit is greater
            max_digit = digit
        n //= 10  # Remove the last digit
    return max_digit

# Example usage:
print(max_digit(8731))  # Output: 8
print(max_digit(405))    # Output: 5

14. Count Digits Greater than a Given Number

Create a function called count_digits_greater_than that takes a positive integer and a threshold digit as parameters. The function should return the count of digits in the number that are greater than the threshold.

Solution 
def count_digits_greater_than(n, threshold):
    count = 0
    while n > 0:
        digit = n % 10  # Extract the last digit
        if digit > threshold:  # Check if it's greater than the threshold
            count += 1
        n //= 10  # Remove the last digit
    return count

# Example usage:
print(count_digits_greater_than(5423, 4))  # Output: 2 (5 and 5 are greater than 4)
print(count_digits_greater_than(9876, 7))   # Output: 2 (8 and 9 are greater than 7)
