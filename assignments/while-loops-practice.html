Python Practice Exercises - While Loops

Complete the following exercises to strengthen your understanding of While Loops Concepts.

1. Sum of Even Numbers

Write a while loop to find the sum of all even numbers in a given list. 

numbers = [3, 5, 8, 12, 7, 10, 14]
Solution 
numbers = [3, 5, 8, 12, 7, 10, 14]
total_sum = 0
index = 0

while index < len(numbers):
    if numbers[index] % 2 == 0:
        total_sum += numbers[index]
    index += 1

print("Sum of even numbers:", total_sum)

2. Find the Maximum Value

Use a while loop to find the maximum value in a given list of numbers. 

numbers = [23, 89, 12, 67, 34]
Solution 
numbers = [23, 89, 12, 67, 34]
max_value = numbers[0]
index = 1

while index < len(numbers):
    if numbers[index] > max_value:
        max_value = numbers[index]
    index += 1

print("Maximum value:", max_value)

3. Count Negative Numbers in a List

Write a while loop to count the number of negative numbers in a given list. 

numbers = [-2, 4, -3, 5, -6, 7, -1]
Solution 
numbers = [-2, 4, -3, 5, -6, 7, -1]
negative_count = 0
index = 0

while index < len(numbers):
    if numbers[index] < 0:
        negative_count += 1
    index += 1

print("Count of negative numbers:", negative_count)

4. Collatz Conjecture

Implement the Collatz Conjecture: given a number, if it's even, divide it by 2, and if it's odd, multiply it by 3 and add 1. Repeat this process until n becomes 1. 

number = 6
Solution 
n = int(input("Enter a number: "))

while n != 1:
    print(n, end=" ")
    if n % 2 == 0:
        n = n // 2
    else:
        n = n * 3 + 1

print(n)

5. Count the Digits in a Number

Write a while loop to count the number of digits in a given integer. For example, the number 7890 should result in a count of 4 digits. 

number = 7890
Solution 
number = 7890
digit_count = 0

while number > 0:
    number //= 10
    digit_count += 1

print("Number of digits:", digit_count)

6. Find the Minimum Digit in a Number

Write a while loop to find the smallest digit in a given number. For example, if the number is 57321, the minimum digit should be 1. 

number = 57321
Solution 
number = 57321
min_digit = 9  # Start with the highest possible single digit

while number > 0:
    digit = number % 10  # Extract the last digit
    if digit < min_digit:
        min_digit = digit  # Update min_digit if a smaller digit is found
    number //= 10  # Remove the last digit

print("Minimum digit:", min_digit)

7. Count Even Digits Greater Than 5

Write a while loop that counts the number of even digits in a given number that are greater than 5. For example, for the number 28675, the result should be 2 (counting 6 and 8).

Solution 
number = 28675 
count = 0
while number > 0: 
    digit = number % 10 
    if digit % 2 == 0 and digit > 5: 
        count += 1 
    number //= 10
    
print("Count of even digits greater than 5:", count)

9. Cumulative Sum of a List

Write a while loop that calculates the cumulative sum of numbers in a list and prints the cumulative sum in each iteration.

Solution 
numbers = [2, 3, 5, 7]
index = 0
current_sum = 0

while index < len(numbers):
    current_sum += numbers[index]
    print("Cumulative sum: ", current_sum)
    index += 1

print("Final cumulative sum:", current_sum)

10. Palindrome Checker (List of Numbers)

Write a while loop that uses the two-pointer technique to check if a given list of numbers is a palindrome. A palindrome reads the same forward and backward, like [1, 2, 3, 2, 1]. 

numbers = [1, 2, 3, 2, 1]
Solution 
numbers = [1, 2, 3, 2, 1]
is_palindrome = True
left = 0
right = len(numbers) - 1

while left < right:
    if numbers[left] != numbers[right]:
        is_palindrome = False
        break
    left += 1
    right -= 1

print("Is palindrome:", is_palindrome)
