1. Determine Zero, Odd or Even

Write a program that takes a number from the user and determines if it is zero, odd, or even. Print the result.

Solution 
number = int(input("Enter a number: "))
if number == 0:
    print("The number is zero")
elif number % 2 == 0:
    print("The number is even")
else:
    print("The number is odd")

2. Grade Classification

Write a program that reads a score from the user and classifies it into grades: 'A' for scores 90 and above, 'B' for scores between 80 and 89, 'C' for scores between 70 and 79, and 'F' for scores below 70. Print the grade.

Solution 
score = int(input("Enter your score: "))
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")

3. Check Leap Year

Write a program that takes a year from the user and checks if it is a leap year. A year is a leap year if it is divisible by 4, but not by 100, unless it is also divisible by 400. Print whether the year is a leap year or not.

Solution 
year = int(input("Enter a year: "))
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print("The year is a leap year")
else:
    print("The year is not a leap year")

4. Generate Multiples of a Number

Write a program that takes a number from the user and prints its first 10 multiples using a `for` loop.

Solution 
number = int(input("Enter a number: "))
for i in range(1, 11):
    print(number * i)

5. Sum of Even Numbers

Write a program that calculates the sum of all even numbers between 1 and a user-defined number using a `for` loop.

Solution 
limit = int(input("Enter a limit: "))
sum_even = 0
for i in range(2, limit + 1, 2):
    sum_even += i
print("Sum of even numbers:", sum_even)

6. Fibonacci Sequence

Write a program that prints the first 10 numbers of the Fibonacci sequence using a `for` loop.

Solution 
a, b = 0, 1
print(a, end=' ')
for _ in range(9):
    a, b = b, a + b
    print(a, end=' ')
print()

7. Print a Pyramid Pattern

Write a program that prints a pyramid pattern of stars. The number of rows should be taken from the user.

Solution 
rows = int(input("Enter number of rows: "))
for i in range(1, rows + 1):
    print(' ' * (rows - i) + '*' * (2 * i - 1))

8. Count Vowels in a String

Write a program that counts the number of vowels in a user-provided string.

Solution 
text = input("Enter a string: ")
vowels = 'aeiouAEIOU'
count = 0
for char in text:
    if char in vowels:
        count += 1
print("Number of vowels:", count)
