Overview

This week, we will explore control flow in programming, focusing on `if/else` statements and the basics of `for` loops in Python.

If/Else Statements

Conditional statements allow you to execute certain parts of your code based on specific conditions. The `if/else` structure in Python is one of the fundamental tools for decision-making in code.

Basic structure:

if condition:
    # Code to execute if the condition is True
else:
    # Code to execute if the condition is False

Example:

age = 20

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

In this example, the program checks if the value of `age` is greater than or equal to 18. If true, it prints "You are an adult." Otherwise, it prints "You are a minor."

Elif and Else Statements

Python also allows you to chain multiple conditions using `elif` (short for "else if"). This is useful when you want to check several conditions in sequence.

Example:

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")

Here, the program checks the value of `score` and prints the corresponding grade. If none of the conditions are met, it defaults to "Grade: F".

For Loops

`For` loops in Python are used to iterate over a sequence (such as a list, tuple, or string) and execute a block of code for each item in the sequence.

Basic structure:

for item in sequence:
    # Code to execute for each item

Example:

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

In this example, the `for` loop iterates over the `fruits` list and prints each item. The output will be:

apple
banana
cherry

In addition to iterating over lists, `for` loops can also use the `range()` function to generate a sequence of numbers. The `range()` function is commonly used for counting iterations in a loop.

Basic usage of `range()`:

for i in range(start, stop, step):
    # Code to execute for each number in the range

Here, `start` is the starting value (inclusive), `stop` is the ending value (exclusive), and `step` is the increment. If `step` is omitted, it defaults to 1.

Example with `range()`:

for i in range(1, 6):
    print(i)

This loop will print the numbers from 1 to 5:

1
2
3
4
5

You can also use `range()` to create more complex loops. For example, to print squares of numbers from 1 to 5:

for i in range(1, 6):
    print(i ** 2)

This will print the squares of numbers from 1 to 5:

1
4
9
16
25

Practice

Try out these exercises to reinforce your understanding:

1. Simple If/Else Statement

Write an `if/else` statement that checks if a number is positive or negative, and prints a message accordingly.

Solution 
number = -5

if number >= 0:
    print("The number is positive.")
else:
    print("The number is negative.")

2. Elif Statement

Write a program that categorizes a person's age into "Child," "Teenager," "Adult," or "Senior" based on their age.

Solution 
age = 65

if age < 13:
    print("Child")
elif 13 <= age < 20:
    print("Teenager")
elif 20 <= age < 65:
    print("Adult")
else:
    print("Senior")

3. For Loop

Create a `for` loop that prints the square of each number in a list.

Solution 
numbers = [1, 2, 3, 4, 5]

for number in numbers:
    print(number ** 2)

4. Factorial Calculation

Write a program that calculates the factorial of a number provided by the user using a `for` loop.

Solution 
number = int(input("Enter a number: "))
factorial = 1
for i in range(1, number + 1):
    factorial *= i
print("Factorial:", factorial)
