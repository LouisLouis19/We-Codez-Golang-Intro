Overview

This week, we dive deeper into Python functions by learning about function parameters, scope, and using built-in functions. Understanding these concepts is essential for writing clean, reusable code.

Function Parameters

Function parameters allow you to pass data into a function, making it more flexible and reusable. You define parameters in the function's definition and pass arguments to them when calling the function.

Example:

def greet(name):
    print("Hello, " + name)

greet("Alice")

Here, name is a parameter, and "Alice" is the argument passed to it when the function is called.

Default Parameters

In Python, you can assign default values to parameters. This means that if an argument is not provided for that parameter when the function is called, the default value will be used.

Example:

def greet(name, greeting="Hello"):
    print(greeting + ", " + name)

greet("Alice")
greet("Bob", "Hi")

In the above example, greeting has a default value of "Hello". When greet("Alice") is called, the default greeting is used. When greet("Bob", "Hi") is called, the default greeting is overridden by "Hi".

Scope

Scope refers to the visibility of variables within different parts of a program. In Python, variables defined inside a function are local to that function and cannot be accessed outside of it.

Example:

def my_function():
    x = 10  # Local variable
    print(x)

my_function()
print(x)  # This will cause an error because x is not accessible here

In the above example, the variable x is local to my_function() and cannot be used outside of it.

Using Built-in Python Functions

Python comes with a number of built-in functions that you can use to perform common tasks. These functions are always available and do not require any additional code to use.

Examples of built-in functions:

  • len(): Returns the length of an object.
  • max(): Returns the largest item in an iterable.
  • min(): Returns the smallest item in an iterable.
  • sum(): Sums the items of an iterable.
  • type(): Returns the type of an object.

Example:

numbers = [1, 2, 3, 4, 5]
print("Length:", len(numbers))
print("Max:", max(numbers))
print("Min:", min(numbers))
print("Sum:", sum(numbers))
print("Type:", type(numbers))

4. Modify a Function to Use Parameters

Modify the greet() function so that it takes two parameters: name and age. The function should print a message that includes both the name and age.

Solution 
def greet(name, age):
    print("Hello, " + name + ". You are " + str(age) + " years old.")

greet("Alice", 25)

5. Using Scope

Create a function calculate_area that calculates the area of a rectangle. Define variables for length and width inside the function and return the area. Ensure that these variables are not accessible outside the function.

Solution 
def calculate_area():
    length = 5
    width = 3
    return length * width

area = calculate_area()
print(area)
