Overview

This week, we dive deeper into Python functions by learning about function parameters, scope, and using built-in functions. Understanding these concepts is essential for writing clean, reusable code.

Function Parameters

Function parameters allow you to pass data into a function, making it more flexible and reusable. You define parameters in the function's definition and pass arguments to them when calling the function.

Example:

def greet(name):
    print("Hello, " + name)

greet("Alice")

Here, name is a parameter, and "Alice" is the argument passed to it when the function is called.

Default Parameters

In Python, you can assign default values to parameters. This means that if an argument is not provided for that parameter when the function is called, the default value will be used.

Example:

def greet(name, greeting="Hello"):
    print(greeting + ", " + name)

greet("Alice")
greet("Bob", "Hi")

In the above example, greeting has a default value of "Hello". When greet("Alice") is called, the default greeting is used. When greet("Bob", "Hi") is called, the default greeting is overridden by "Hi".

Using Multiple Default Parameters

You can assign default values to multiple parameters in a function. However, when you do so, all parameters with default values must come after the parameters that do not have default values in the function definition.

Example:

def describe_pet(pet_name, animal_type="dog", sound="woof"):
    print("I have a " + animal_type + " named " + pet_name + ".")
    print("It says " + sound + "!")

describe_pet("Buddy")
describe_pet("Milo", "cat", "meow")
describe_pet("Charlie", sound="chirp")

In this example, animal_type and sound have default values. When describe_pet("Buddy") is called, the default values are used. When describe_pet("Milo", "cat", "meow") is called, both defaults are overridden. When describe_pet("Charlie", sound="chirp") is called, only the sound parameter is overridden, while the default animal_type is used.

Importance of Default Parameters

Default parameters are useful when you want to provide flexible function interfaces, allowing users to call the function with fewer arguments while still having the option to override defaults if needed. This can make your functions more versatile and easier to use in different scenarios.

Considerations

- Default parameters are evaluated only once when the function is defined, not each time the function is called. This is particularly important when using mutable objects (like lists or dictionaries) as default values.

Example:

def append_to_list(value, my_list=[]):
    my_list.append(value)
    return my_list

print(append_to_list(1))
print(append_to_list(2))
print(append_to_list(3))

The above function might not behave as expected because the default list is shared across all calls to the function, leading to cumulative changes.

Output:

[1]
[1, 2]
[1, 2, 3]

To avoid this, use None as the default value and initialize the list inside the function: 

def append_to_list(value, my_list=None):
    if my_list is None:
        my_list = []
    my_list.append(value)
    return my_list

print(append_to_list(1))
print(append_to_list(2))
print(append_to_list(3))

Output:

[1]
[2]
[3]

Now, each call to the function creates a new list, avoiding unintended side effects.

Scope

Scope refers to the visibility and lifetime of variables within different parts of a program. In Python, variables can have different levels of scope:

  • Local Scope: Variables defined inside a function, which are accessible only within that function.
  • Global Scope: Variables defined outside any function, which are accessible from any part of the program.

In general, variables created inside a function are local to that function and cannot be accessed outside of it. This prevents other parts of the program from unintentionally modifying those variables.

Local Scope

Example:

def my_function():
    x = 10  # Local variable
    print(x)  # Accessible inside the function

my_function()
print(x)  # This will cause an error because x is not accessible here

In the above example, the variable x is local to my_function() and cannot be used outside of it. Attempting to access it outside the function will cause a NameError.

Global Scope

Variables defined outside of a function have global scope and can be accessed from anywhere within the program.

Example:

x = 20  # Global variable

def my_function():
    print(x)  # Accessing the global variable

my_function()
print(x)  # Still accessible here

In this case, x is a global variable and can be accessed both inside and outside my_function().

Modifying Global Variables

If you want to modify a global variable inside a function, you need to use the global keyword. Without it, Python will treat any variable assignment inside the function as a new local variable.

Example:

x = 20  # Global variable

def my_function():
    global x
    x = 30  # Modify the global variable
    print(x)

my_function()
print(x)  # Global variable x has been changed

By using the global keyword, you tell Python to modify the global variable instead of creating a local one.

Summary

Understanding scope in Python is crucial for managing variable visibility and avoiding conflicts or unintended side effects in your programs. Local, global, and enclosing (with nonlocal) scopes are important concepts when working with functions and nested functions.

Using Built-in Python Functions

Python comes with a wide range of built-in functions that perform common tasks, simplifying code and making it more readable. These functions are always available without the need to import any modules, meaning you can start using them right away.

Common Built-in Functions

Here are some commonly used built-in Python functions:

  • len(): Returns the length (number of items) of an object (e.g., list, tuple, string).
  • max(): Returns the largest item in an iterable or the largest of multiple arguments.
  • min(): Returns the smallest item in an iterable or the smallest of multiple arguments.
  • sum(): Returns the sum of all items in an iterable.
  • type(): Returns the type of an object (e.g., list, int, str).
  • abs(): Returns the absolute value of a number.
  • round(): Rounds a floating-point number to a specified number of decimal places.
  • sorted(): Returns a new sorted list from the items in an iterable.

Example Usage

Example of using some common built-in functions:

numbers = [1, 2, 3, 4, 5]
print("Length:", len(numbers))  # Output: 5
print("Max:", max(numbers))     # Output: 5
print("Min:", min(numbers))     # Output: 1
print("Sum:", sum(numbers))     # Output: 15
print("Type:", type(numbers))   # Output: 

# Using other built-in functions
value = -42
print("Absolute Value:", abs(value))   # Output: 42
float_number = 7.56789
print("Rounded Value:", round(float_number, 2))  # Output: 7.57
print("Sorted List:", sorted([4, 2, 3, 1, 5]))   # Output: [1, 2, 3, 4, 5]

Function Details

Here are some additional details on a few of these functions:

  • abs(): This function returns the absolute value of a number, which is the number's distance from zero on the number line, without considering its sign (positive or negative). 
    print(abs(-10))  # Output: 10
print(abs(10))   # Output: 10
  • round(): This function rounds a floating-point number to a specified number of decimal places. If no number of decimal places is provided, it rounds to the nearest integer. 
    print(round(4.567))       # Output: 5
print(round(4.567, 2))    # Output: 4.57
print(round(4.567, 1))    # Output: 4.6
  • sorted(): This function returns a new sorted list from an iterable, such as a list or tuple. It does not modify the original iterable. 
    unsorted_list = [3, 1, 4, 1, 5, 9]
sorted_list = sorted(unsorted_list)
print("Unsorted:", unsorted_list)  # Output: [3, 1, 4, 1, 5, 9]
print("Sorted:", sorted_list)      # Output: [1, 1, 3, 4, 5, 9]

Exploring More Built-in Functions

Python provides over 60 built-in functions, and they can greatly enhance your programming efficiency. Some other commonly used functions include:

  • input(): Reads a string from standard input.
  • int(), float(), str(): Convert between different data types.
  • open(): Opens a file and returns a file object.
  • help(): Provides help on Python objects and functions.

For a full list of built-in functions, you can check Python's official documentation or use the help() function directly in your Python environment.

1. Loop with Default Parameters

Create a function repeat_message that takes a message and a number of repetitions as parameters, with a default value of 3 for repetitions. The function should print the message only on even-numbered repetitions (e.g., 2nd, 4th, etc.).

Solution 
def repeat_message(message, repetitions=3):
    for i in range(1, repetitions + 1):
        if i % 2 == 0:
            print(f"Repetition {i}: {message}")

repeat_message("Hello!")  # Will print only on 2nd and 4th repetitions
repeat_message("Python is fun", 5)

2. Cumulative Product of Odd Numbers

Write a function cumulative_product that takes a list of numbers and returns the cumulative product of only the odd numbers. Ignore even numbers during the calculation.

Solution 
def cumulative_product(numbers):
    product = 1
    for num in numbers:
        if num % 2 != 0:
            product *= num
    return product

numbers = [1, 2, 3, 4, 5]
print(cumulative_product(numbers))  # Output: 15

3. Loop with Conditional Logic and Default Parameters

Create a function find_largest that accepts a list of numbers and an optional threshold (default value of 10). The function should return the largest number in the list that is less than the threshold.

Solution 
def find_largest(numbers, threshold=10):
    largest = None
    for num in numbers:
        if num < threshold:
            if largest is None or num > largest:
                largest = num
    return largest

numbers = [12, 7, 5, 19, 2]
print(find_largest(numbers))  # Output: 7

4. Find the Longest Increasing Sequence

Write a function longest_increasing_sequence that takes a list of numbers and returns the length of the longest increasing sequence in the list. The sequence consists of consecutive numbers that are in increasing order.

Solution 
def longest_increasing_sequence(numbers):
    max_length = 0
    current_length = 1
    for i in range(1, len(numbers)):
        if numbers[i] > numbers[i - 1]:
            current_length += 1
            max_length = max(max_length, current_length)
        else:
            current_length = 1
    return max(max_length, current_length)

numbers = [1, 2, 3, 1, 2, 3, 4, 1]
print(longest_increasing_sequence(numbers))  # Output: 4

5. Loop with Built-in Functions and Reverse Order

Create a function reverse_and_sum that takes a list of numbers, reverses the order of the list, and then sums the reversed numbers using the sum() built-in function.

Solution 
def reverse_and_sum(numbers):
    reversed_numbers = numbers[::-1]
    return sum(reversed_numbers)

numbers = [1, 2, 3, 4, 5]
print(reverse_and_sum(numbers))  # Output: 15
